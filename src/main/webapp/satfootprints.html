/satfootprints

Returns satellite footprints as polygon. Foot prints crossing 180 lon are excluded for now.

Parameters

  • f : Format (json, geojson, txt (default))
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.

Formats

  • json: JSON Array
  • ndjson: Newline Delimited JSON. Each line is a valid JSON object
  • geojson: Valid GeoJSON
  • txt: Pipe delimited values. Each event is new line delimited

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • geometry: polygon
  • npts: Number of points in the polygon (20 - 500) defaults to 50.
  • t: Epoch Time to return

Examples

  • Delimited
    • Get all satellites: /satfootprints
    • Get satellites by number: /satfootprints?nums=29709,43575
  • JSON
    • Get all satellites: /satfootprints?f=json
    • Get satellites by number: /satfootprints?f=json&nums=29709,43575
    • Get satellites by number using 100 points: /satfootprints?f=json&nums=29709,43575&npts=100
  • NDJSON
    • Get all satellites: /satfootprints?f=ndjson
    • Get satellites by number: /satfootprints?f=ndjson&nums=29709,43575
    • Get satellites by number using 100 points: /satfootprints?f=ndjson&nums=29709,43575&npts=100
  • GEOJSON
    • Get all satellites: /satfootprints?f=geojson
    • Get satellites by name: /satfootprints?f=geojson&names=ISS (ZARYA)
    • Get satellites by number: /satfootprints?f=geojson&nums=29709,43575
