/sattracks

Returns line geometries

Parameters

  • f : Format (json, ndjson, geojson, txt (default))
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.
  • st: Epoch Time to start default to current time in seconds or milliseconds
  • td: Length of time for the line returned in seconds.
  • ts: Time step seconds.

Formats

  • json: JSON Array
  • ndjson: Newline Delimited JSON. Each line is a valid JSON object
  • geojson: Valid GeoJSON
  • txt: Pipe delimited values. Each event is new line delimited

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • geometry: Line

Examples

  • Delimited
    • Get all satellites: /sattracks
    • Get satellites by number: /sattracks?nums=29709,41895
  • JSON
    • Get all satellites: /sattracks?f=json
    • Get satellites by number: /sattracks?f=pjson&nums=29709,41895
    • Get satellites by number for 300 seconds with 30 second samples: /sattracks?f=pjson&nums=29709,41895&td=300&ts=30
  • NDJSON
    • Get all satellites: /sattracks?f=ndjson
    • Get satellites by number: /sattracks?f=ndpjson&nums=29709,41895
    • Get satellites by number for 300 seconds with 30 second samples: /sattracks?f=pjson&nums=29709,41895&td=300&ts=30
  • GEOJSON
    • Get all satellites: /satellites?f=geojson
    • Get satellites by name: /sattracks?f=geojson&names=GPS BIIR-3 (PRN 11),GPS BIIR-6 (PRN 14)
    • Get satellites by number: /sattracks?f=geojson&nums=29709,41895
