/ellipses

Returns random small ellipses

Parameters

  • f : Format (json, geojson, txt (default))
  • num: Number to return (defaults to 1)

Formats

  • json: JSON Array
  • ndjson: Newline Delimited JSON. Each line is a valid JSON object
  • geojson: Valid GeoJSON
  • txt: Pipe delimited values. Each event is new line delimited

Fields in Order Returned

  • a: Semi-Major Axis
  • b: Semi-Minor Axis
  • rot: Rotation from north
  • num: One up number
  • clon: Center Longitude
  • clon: Center Longitude

Examples

  • Delimited
    • Get one ellipse /ellipses
    • Get 20 ellipses /ellipses?num=20
  • JSON
    • Get one ellipse /ellipses?f=json
    • Get 20 ellipses /ellipses?f=json&num=20
  • NDJSON
    • Get one ellipse /ellipses?f=ndjson
    • Get 20 ellipses /ellipses?f=ndjson&num=20
  • GEOJSON
    • Get one ellipse /ellipses?f=geojson
    • Get 20 ellipses /ellipses?f=geojson&num=20
