Websats

Websats returns satellite information.

It was created to produce test data and is not intended for production use.

  • /satellites : returns satellite positions as points
  • /sattracks : returns satellite tracks as lines
  • /satfootprints : returns satellite footprints as polygons
  • /tles : returns satellite two line elements sets

WebSocket for Satellites

Returns all satellites every 10 seconds. Geometries converted to Web Mercator to support mapping.

  • WebSocket Endpoint: /websats/SatStream/subscribe
  • Results in Window: /satstream.html
  • Plot Results on Map: /satstreamjs.html

Unrelated to satellites

  • /ellipses : returns random ellipses
