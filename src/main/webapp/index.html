Websats

Websats returns satellite locations. It was created to produce test data and is not intended for production use.

Rest API

/satellites

Returns point geometries

Parameters

  • f : Format (json, geojson; unset or any other value returns txt
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.
  • t: Epoch Time to return

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • timestamp: Number of seconds from Epoch
  • dtg: Date and time of the location. DD-MMM-YYYY hh:mm:ss.ss
  • lon: Longitude of Satellite
  • lat: Latitude of Satellite
  • alt: Altitude of Satellite

Examples

  • Pipe Delimited
    • Get all satellites: /satellites
    • Get satellites by number: /satellites?nums=29709,41895
  • JSON
    • Get all satellites: /satellites?f=json
    • Get satellites by number: /satellites?f=json&nums=29709,41895
  • GEOJSON
    • Get all satellites: /satellites?f=geojson
    • Get satellites by name: satellites?f=geojson&names=GPS BIIR-3 (PRN 11),GPS BIIR-6 (PRN 14)

/sattracks

Returns line geometries

Parameters

  • f : Format (txt, json, geojson
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.
  • st: Epoch Time to start default to current time in seconds or milliseconds
  • td: Length of time for the line returned in seconds.
  • ts: Time step seconds.

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • geometry: Line

Examples

  • Pipe Delimited
    • Get all satellites: /sattracks
    • Get satellites by number: /sattracks?nums=29709,41895
  • JSON
    • Get all satellites: /sattracks?f=json
    • Get satellites by number: /sattracks?f=pjson&nums=29709,41895
    • Get satellites by number for 300 seconds with 30 second samples: /sattracks?f=pjson&nums=29709,41895&td=300&ts=30
  • GEOJSON
    • Get all satellites: /satellites?f=geojson
    • Get satellites by name: /sattracks?f=geojson&names=GPS BIIR-3 (PRN 11),GPS BIIR-6 (PRN 14)
    • Get satellites by number: /sattracks?f=geojson&nums=29709,41895

/satfootprints

Returns satellite footprints as polygon. Foot prints crossing 180 lon are excluded for now.

Parameters

  • f : Format (txt, json, geojson
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • geometry: polygon
  • npts: Number of points in the polygon (20 - 500) defaults to 50.
  • t: Epoch Time to return

Examples

  • Pipe Delimited
    • Get all satellites: /satfootprints
    • Get satellites by number: /satfootprints?nums=29709,41895
  • JSON
    • Get all satellites: /satfootprints?f=json
    • Get satellites by number: /satfootprints?f=pjson&nums=29709,41895
    • Get satellites by number using 100 points: /satfootprints?f=json&nums=29709,41895&npts=100
  • GEOJSON
    • Get all satellites: /satfootprints?f=geojson
    • Get satellites by name: /satfootprints?f=geojson&names=ISS (ZARYA)
    • Get satellites by number: /satfootprints?f=geojson&nums=29709,41895

/tles

Returns point geometries

Parameters

  • f : Format (json, txt; unset or any other value returns txt
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.

Lines Returned for each Satellite

  • TLE Header
  • TLE Line 1
  • TLE Line 2

Examples

  • Text TLE
    • TLE for all satellites: /tles
    • TLE for satellites by name: /tles?nums=29709,41895
  • JSON
    • TLE all satellites: /tles?f=json
    • TLE for satellites by number: /tles?f=json&names=GPS*
