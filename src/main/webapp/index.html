Websats

Websats returns satellite locations. It was created to produce test data and is not intended for production use.

Fields in CSV order

  • name: Satellite Name
  • num: Satellite Number
  • timestamp: Number of seconds from Epoch
  • dtg: Date and time of the location. DD-MMM-YYYY hh:mm:ss
  • lon: Longitude of Satellite
  • lat: Latitude of Satellite
  • alt: Altitude of Satellite

Rest API

  • Get all satellites (default csv): /satellites
  • Get all satellites as json: /satellites?f=json
  • Get all satellites as pretty json: /satellites?f=pjson
  • Get specific satellites by name (csv): /satellites?names=GPS BIIR-3 (PRN 11),GPS BIIR-6 (PRN 14)
  • Get specific satellites by number (pjson): /satellites?f=pjson&nums=29709,41895
