Websats

Websats returns satellite locations. It was created to produce test data and is not intended for production use.

Rest API

/satellites

Returns point geometries

  • f : Format (json, pjson, geojson; unset or any other value returns csv
  • names: Comma separated list of Satellite Names (They have to be exactly as listed)
  • nums: Comma separated list of Satellite Nums
  • t: Epoch Time to return

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • timestamp: Number of seconds from Epoch
  • dtg: Date and time of the location. DD-MMM-YYYY hh:mm:ss.ss
  • lon: Longitude of Satellite
  • lat: Latitude of Satellite
  • alt: Altitude of Satellite

Examples

  • Pipe Delimited
    • Get all satellites: /satellites
    • Get all satellites: /satellites?nums=29709,41895
  • JSON
    • Get all satellites: /satellites?f=json
    • Get specific satellites by number: /satellites?f=json&nums=29709,41895
  • GEOJSON
    • Get all satellites: /satellites?f=geojson
    • Get specific satellites by name: satellites?f=geojson&names=GPS BIIR-3 (PRN 11),GPS BIIR-6 (PRN 14)

/sattracks

Returns line geometries

  • f : Format (txt, json, geojson
  • names: Comma separated list of Satellite Names (defaults to all satellites)
  • nums: Comma separated list of Satellite Nums (defaults to all satellites)
  • st: Epoch Time to start default to current time in seconds or milliseconds
  • td: Length of time for the line returned in seconds.
  • ts: Time step seconds.

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • geometry: Line

Examples

  • Pipe Delimited
    • Get all satellites: /sattracks
    • Get all satellites: /sattracks?nums=29709,41895
  • JSON
    • Get all satellites: /sattracks?f=json
    • Get specific satellites by number: /sattracks?f=pjson&nums=29709,41895
    • Get specific satellites by number for 300 seconds with 30 second samples: /sattracks?f=pjson&nums=29709,41895&td=300&ts=30
  • GEOJSON
    • Get all satellites: /satellites?f=geojson
    • Get specific satellites by name: /sattracks?f=geojson&names=GPS BIIR-3 (PRN 11),GPS BIIR-6 (PRN 14)
    • Get specific satellites by number: /sattracks?f=geojson&nums=29709,41895

/satfootprints

Returns satellite footprints as polygon. Foot prints crossing 180 lon are excluded for now.

  • f : Format (txt, json, geojson
  • names: Comma separated list of Satellite Names (defaults to all satellites)
  • nums: Comma separated list of Satellite Nums (defaults to all satellites)

Fields in Order Returned

  • name: Satellite Name
  • num: Satellite Number
  • geometry: polygon
  • npts: Number of points in the polygon (20 - 500) defaults to 50.
  • t: Epoch Time to return

Examples

  • Pipe Delimited
    • Get all satellites: /satfootprints
    • Get all satellites: /satfootprints?nums=29709,41895
  • JSON
    • Get all satellites: /satfootprints?f=json
    • Get specific satellites by number: /satfootprints?f=pjson&nums=29709,41895
    • Get specific satellites by number using 100 points: /satfootprints?f=json&nums=29709,41895&npts=100
  • GEOJSON
    • Get all satellites: /satfootprints?f=geojson
    • Get specific satellites by name: /satfootprints?f=geojson&names=ISS (ZARYA)
    • Get specific satellites by number: /satfootprints?f=geojson&nums=29709,41895
