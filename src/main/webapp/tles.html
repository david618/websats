/tles

Returns point geometries

Parameters

  • f : Format (json, txt; unset or any other value returns txt
  • names: Comma separated list of Satellite Names. You can end with * to get Satellites staring with prefix (e.g. GPS*).
  • nums: Comma separated list of Satellite Nums. If Nums is specified Names are ignored.

Lines Returned for each Satellite

  • TLE Header
  • TLE Line 1
  • TLE Line 2

Examples

  • Text TLE
    • TLE for all satellites: /tles
    • TLE for satellites by name: /tles?nums=29709,41895
  • JSON
    • TLE all satellites: /tles?f=json
    • TLE for satellites by number: /tles?f=json&names=GPS*
